# Security Policy

Please report security issues privately instead of opening a public issue.

Email: mark@builderproapps.com

Useful reports include:

- Crash or memory corruption bugs in native parsing, serialization, rendering, or PDF/background handling.
- Malformed notebook payloads that can corrupt saved data or crash the app.
- Native bridge issues that expose unintended file access or unsafe callbacks.
- Dependency or packaging issues that could compromise downstream apps.
